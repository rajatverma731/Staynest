# 🏡 StayNest

StayNest is a full-stack web application that allows users to explore, list, and manage rental properties seamlessly.

---

## 🚀 Features

* 🔐 User Authentication (Login/Signup)
* 🏠 Add, Edit, Delete Property Listings
* 📍 Location-based Listings
* 🖼️ Image Upload Support
* 💬 Interactive UI with dynamic data

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)

---

## 📁 Project Structure

```
init/
models/
public/
utils/
views/
app.js
```

---

## ⚙️ Installation

```bash
git clone https://github.com/rajatverma731/Staynest.git
cd Staynest
npm install
node app.js
```
